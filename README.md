# jakebouma.com

Personal website and portfolio for Jake Bouma. Built with Next.js and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Before Deploying

1. **Add your headshot**: Place your headshot image at `public/images/headshot.jpg`
2. Review all content in the page files under `src/app/`

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new) and import the repo
3. Vercel will auto-detect Next.js — just click Deploy
4. Set your custom domain (jakebouma.com) in Project Settings → Domains
